README: Archive Search Canine

Überblick
"Truffle_DOG" ist eine Desktop-Anwendung zum Durchsuchen von XML-Dateien nach Personen,
die an einem spezifischen Datum geboren wurden.

Funktionen
- Suche nach Geburtsdatum in XML-Dateien.
- Auswahl eines Verzeichnisses für die Suche.
- Anzeige der Ergebnisse in der Anwendung.

Nutzung
- Anwendung öffnen.
- Geburtsdatum (Tag, Monat, Jahr) eingeben.
- Verzeichnis für die Suche auswählen.
- Auf "Suchen" klicken.
- Ergebnisse werden angezeigt.

Voraussetzungen
Python und Tkinter müssen installiert sein.
